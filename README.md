# Superpowers Skills Library

A collection of 14 Claude Code skills for TDD, debugging, collaboration patterns, and proven development techniques.

## Installation

This repository can be used as a Claude Code plugin:

1. Clone this repository
2. Install as a local plugin in Claude Code

## Skills

| Skill | Description |
|-------|-------------|
| brainstorming | Explores user intent, requirements and design before implementation |
| dispatching-parallel-agents | Coordinate multiple agents for independent tasks |
| executing-plans | Execute implementation plans with review checkpoints |
| finishing-a-development-branch | Guide completion of development work |
| receiving-code-review | Handle code review feedback with technical rigor |
| requesting-code-review | Request code review to verify work meets requirements |
| subagent-driven-development | Execute implementation plans with subagents |
| systematic-debugging | Debug with methodical root cause analysis |
| test-driven-development | Implement features using TDD |
| using-git-worktrees | Create isolated git worktrees for feature work |
| using-superpowers | Introduction to finding and using skills |
| verification-before-completion | Verify work before claiming completion |
| writing-plans | Create detailed implementation plans |
| writing-skills | Create and edit skills |
